== Description ==

Disables all WordPress auto-formatting automatically. Similar to the popular RawHTML plugin, but applies the effect to all posts & pages automatically.

== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What is this for? =

To remove various types of auto-formatting that WordPress adds to your content and excerpts.
